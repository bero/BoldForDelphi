ComboBox

Bold-aware dropdown selection control

Overview

The TBoldComboBox component provides dropdown selection bound to Bold object lists. It automatically populates from a handle and updates when data changes.

Features

  • Populate from Bold object collections
  • OCL expression for display text
  • Automatic refresh on data changes
  • Support for null selection
