SQL Handle

Execute custom SQL within Bold

Overview

SQL handles allow direct SQL execution while maintaining Bold object consistency. Use for complex queries that are difficult to express in OCL or for performance optimization.

Features

  • Direct SQL - Write database-specific queries
  • Object mapping - Results mapped to Bold objects
  • Parameterized - Prevent SQL injection with parameters
  • Read/Write - Both SELECT and modification statements
