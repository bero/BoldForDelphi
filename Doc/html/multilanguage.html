MultiLanguage

Localize your Bold applications

Overview

The MultiLanguage example shows how to implement internationalization in Bold applications. Display text, labels, and messages can be translated to different languages.

Features

  • Language switching - Change display language at runtime
  • Attribute translations - Localized attribute names
  • UI localization - Translated form captions and labels
  • Resource-based - External language resource files
