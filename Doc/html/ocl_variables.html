OCL Variables

Pass parameters to OCL expressions

Overview

OCL variables allow you to pass external values into OCL expressions. This enables parameterized queries and dynamic filtering based on runtime values.

Features

  • Parameter binding - Pass Delphi values to OCL
  • Dynamic queries - Build queries with runtime parameters
  • Type safety - Variables are typed and validated
  • Reusable expressions - Same OCL with different parameters
