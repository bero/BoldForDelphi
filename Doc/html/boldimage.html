BoldImage

Display and edit images stored in Bold objects

Overview

The TBoldImage component displays and edits images stored as BLOB attributes in Bold objects. It supports common image formats and provides load/save functionality.

Features

  • Display images from Bold BLOB attributes
  • Load images from file
  • Auto-stretch or center images
  • Automatic updates when data changes
