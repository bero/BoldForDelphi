LogBridge

Interface-based logging with pluggable sinks

Overview

LogBridge demonstrates Bold's logging infrastructure with pluggable log sinks for debugging and monitoring application behavior.

How It Works

Log messages from Bold components flow through the LogBridge to configurable output sinks:

Features

  • Multiple log sources - ObjectSpace, Persistence, SQL Executor, Subscription system
  • Configurable levels - Error, Warning, Info, Debug filtering
  • Pluggable sinks - File, Memo, Console, Database, or custom implementations
  • Interface-based - Implement ILogSink for custom logging targets
