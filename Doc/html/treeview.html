TreeView

Hierarchical data display with Bold TreeView

Overview

The TBoldTreeView component displays Bold objects in a hierarchical tree structure. It automatically updates when the underlying data changes and supports drag-and-drop, custom icons, and node renderers.

Features

Automatic Updates

Tree refreshes when objects are added, modified, or deleted

Hierarchical Navigation

Navigate object relationships as tree branches

Drag & Drop

Rearrange objects by dragging nodes between branches

Custom Icons

Different icons for different object types or states

Configuration

Key properties for TBoldTreeView:

  • BoldHandle - The root handle providing objects to display
  • NodeDescriptions - Define how each class appears in the tree
  • AutoExpand - Whether to expand nodes automatically
  • HideRootNodes - Show children of root directly

Node Descriptions

Each node type is configured with a TBoldNodeDescription that specifies:

  • Name - OCL expression for the node text
  • IconIndex - Image index for the node icon
  • Children - OCL expression for child objects
  • ContextTypeName - The Bold class this description applies to
