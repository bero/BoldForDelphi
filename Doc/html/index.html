Bold for Delphi Examples

Learn Bold through practical demonstrations

Welcome

Explore the Bold for Delphi framework through these examples. Each example demonstrates specific features and includes complete Delphi projects you can build and run.

Examples are organized into Simple examples that focus on individual features, and Demo examples that demonstrate complete applications.

Key Features

Model-Driven Development

Design your domain in UML, Bold generates Delphi classes with full persistence support

Automatic UI Synchronization

Changes to objects instantly update all bound UI controls - no manual refresh needed

OCL Expression Language

Powerful Object Constraint Language for queries, derived values, and business rules

Derived Attributes

Define calculated properties that automatically update when dependencies change

Built-in Undo/Redo

Full transaction support with unlimited undo levels and checkpoint management

Object Identity

Each database row maps to exactly one object instance - guaranteed consistency

Optimistic Locking

Safe multi-user access with automatic conflict detection and resolution

Multi-Database Support

Works with InterBase, MS SQL, Oracle, PostgreSQL, SQLite and more

  2. Browse examples - Use the navigation menu to find examples by category.
  3. Open in Delphi - Each example page links to the corresponding .dpr file.
  4. Build and run - Compile the project and experiment with the features.
