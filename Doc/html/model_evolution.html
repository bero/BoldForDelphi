Model Evolution

Evolve your model while preserving data

Overview

Model evolution demonstrates how to change your Bold model (add classes, rename attributes, change relationships) while preserving existing data in the database.

Evolution Operations

  • Add class - New table created automatically
  • Add attribute - New column added with default value
  • Rename - Data preserved, only metadata changes
  • Change type - Data converted where possible
  • Remove - Clean up with data migration

Evolution Process

Bold compares the old and new model, generates migration scripts, and preserves your data:

How to Use

  1. Make changes to your Bold model (add/rename/remove classes and attributes)
  2. Bold detects differences between model and database schema
  3. Review the generated migration script
  4. Execute evolution to update database while preserving data
