Custom Attributes

Create your own attribute types

Overview

Custom attributes allow you to extend Bold with your own data types. Define how values are stored, displayed, and validated beyond the built-in types.

Custom Attribute Types

  • Value mapping - Custom serialization to database
  • Display formatting - Control how values appear in UI
  • Validation - Type-specific validation rules
  • Operators - Custom comparison and arithmetic
