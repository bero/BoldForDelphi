Derived Handle

Dynamic handles computed from OCL expressions

Overview

Derived handles provide a way to create object lists dynamically using OCL expressions. The list updates automatically when the underlying data changes.

Use Cases

  • Filtered views - Show subset of objects matching criteria
  • Computed collections - Navigate relationships dynamically
  • Aggregations - Combine data from multiple sources
  • Live updates - Results refresh automatically
  • Unlimited chaining - Chain any number of handles together for complex derivations
